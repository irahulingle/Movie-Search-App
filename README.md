# 🎬 Movie Search App

A simple React application to search for movies using the OMDb API.

## 🔍 Features
- Search for movies by title
- View movie posters, release year, and type
- Responsive design with basic styling

## 🚀 Tech Stack
- React
- OMDb API
- Axios
- Styled-Components
